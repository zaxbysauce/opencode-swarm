## Migration Notes

- `CompactionState` is now per-session. `getCompactionMetrics()` without a `sessionId` argument aggregates across all sessions. `resetCompactionState()` without a `sessionId` clears all sessions.
- `PhaseSchema` now accepts an optional `required_agents` array. Phases without this field default to the global `phase_complete.required_agents` config.
- `buildTaskSummary` and `buildPhaseSummary` now require a `directory` parameter as the first argument.
