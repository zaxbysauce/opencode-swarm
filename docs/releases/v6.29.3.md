# OpenCode Swarm v6.29.3 Release Notes

## Summary

Version **6.29.3** adds a Curator visibility and documentation refresh. `/swarm diagnose` now reports Curator status, and the README/configuration docs were updated to match current Curator behavior and current `.swarm/` artifacts.

---

## Features

### Curator Diagnose Visibility

- Added a `Curator` health check to `/swarm diagnose`.
- Reports whether `curator.enabled` is on.
- Distinguishes disabled, enabled/no-summary, valid summary, corrupt summary, and wrong-schema summary states.

### Documentation Refresh

- Updated the README `.swarm/` tree to include current runtime artifacts.
- Refreshed the README Recent Changes section.
- Added Curator configuration documentation to `docs/configuration.md`.

---

## Upgrade Notes

- Curator remains optional and disabled by default.
- If enabled, Curator writes summary/drift artifacts into `.swarm/`.
- `/swarm diagnose` now gives a quick signal about whether Curator is active and healthy.
