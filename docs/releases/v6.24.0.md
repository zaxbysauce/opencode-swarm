# OpenCode Swarm v6.24.0 Release Notes

## Summary

Version **6.24.0** is a stability and test infrastructure improvement release focused on workflow reliability and test isolation hardening.

---

## Workflow Reliability

### Improved task state management

Bug fixes and edge case handling for task state transitions, reducing the likelihood of tasks becoming stuck in intermediate states during complex multi-agent workflows.

### Error recovery improvements

Enhanced error recovery paths for common failure scenarios in delegation and handoff operations.

---

## Test Infrastructure

### Test isolation hardening

Improved test suite isolation to prevent cross-test contamination, ensuring more reliable CI results and faster failure diagnosis.

### Parallel test execution support

Test infrastructure updates to support parallel test execution without flaky failures.

---

## Migration Notes

- No breaking changes.
- No new configuration options.
- For full commit details, see the [CHANGELOG.md](https://github.com/opencode-ai/opencode-swarm/blob/main/CHANGELOG.md).
