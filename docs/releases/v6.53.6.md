# Release Notes — v6.53.6

## Overview

Patch release fixing null-handling edge cases in the curator knowledge update pipeline. No breaking changes.

## Bug Fixes

### Fixed null/undefined guards in `applyCuratorKnowledgeUpdates`

Added runtime guards for `knowledgeConfig` and `recommendations` inputs in `applyCuratorKnowledgeUpdates` to prevent crashes when null or undefined values are passed. Additionally filters null/undefined items from the recommendations array before processing, ensuring stable behavior across all input shapes.

**Before:** Passing `null` or `undefined` for either parameter would throw or produce undefined behavior.
**After:** Both parameters are now safely guarded; nullish items in the recommendations array are silently filtered out before processing.
