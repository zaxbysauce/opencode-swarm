# OpenCode Swarm v6.25.0 Release Notes

## Summary

Version **6.25.0** is a CI reliability and test alignment release focused on internal tooling improvements.

---

## CI Reliability

### Build pipeline improvements

Resolved issues in CI pipelines that caused intermittent failures, including timing-related race conditions and resource contention.

### Type checking alignment

Fixed inconsistencies between TypeScript compiler settings and runtime type expectations, ensuring typecheck passes consistently in CI.

### Test alignment

Corrected test assertions and fixtures that were misaligned with actual component behavior, eliminating false negative test failures.

- No breaking changes.
- No new configuration options.
