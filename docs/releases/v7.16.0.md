# v7.16.0 - Close skill-review nudge

## Summary

`/swarm finalize` now connects session knowledge to the skill pipeline introduced in v7.11.0.

## Changes

- Close summaries include a lightweight hint when knowledge entries were created during the session, pointing users to `skill_improve` and `skill_generate`.
- Added opt-in `--skill-review` for `/swarm finalize` and deprecated `/swarm close`; it runs `skill_improver` in proposal mode for skills and knowledge.
- Skill review is advisory and bounded. Disabled config, quota exhaustion, service errors, and timeout warnings do not block finalization.
- The default close path remains non-LLM.

## Verification

- Targeted close command tests cover the hint, default no-call behavior, `--skill-review`, `sessionID` propagation, advisory failures, and timeout handling.
